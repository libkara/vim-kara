# vim-kara
Vim syntax highlighting for KARA.

## Installation

Pathogen

```
git clone https://github.com/libkara/vim-kara ~/.vim/bundle/vim-kara
```

vim-plug

```vim
Plug 'libkara/vim-kara'
```

Vundle

```vim
Pluging 'libkara/vim-kara'
```
